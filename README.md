##Introduction
    This project contains automated test scripts written using Playwright(JavaScript) to test the SwiftTranslator('https://www.swifttranslator.com/) website, which converts Singlish text into Sinhala unicode. 

##Test case summary
    A total of 35 test cases have been tested in this project. Pos_Fun(24), Neg_Fun(10), Pos_UI(1).

##How to install
    Run the following commands inside the project folder:
        npm install
        npx playwright install

##How to run tests
    To execute all tests:
        npx playwright test
